# AnyAIHelper
**注意：このリポジトリはまだ開発段階です。公開はされていますが、まだフォークや`[Use this template]`をしないでください。**

**AnyAIProject** は、どんな会話AI（ChatGPT / Claude / Gemini / その他）とも連携できる **半自動プロジェクト支援テンプレート** です。  

## 🧩 テンプレートの使い方
1. このリポジトリの [Use this template] ボタンから新しいリポジトリを作成
2. GitHub Actionsが動作するようにリポジトリのActionsを有効化
3. 最初のイシュー `[Init] プロジェクト概要を作成` を登録
4. 自動生成されたプロンプトファイルを確認

## 使い方
まずは **ai_helper** ブランチに切り替えてください。

`ai_helper/README.md` に詳しい使い方がありますのでそちらをご覧ください。
