# 🧩 AI-Integrated Project Template

**～AIとGitHub Actionsによる半自動プロジェクト立ち上げテンプレート～**

## 📘 概要

このテンプレートリポジトリは、**GitHubの標準機能（Issues / Actions / Branch）とChatGPTなどの会話AIを連携させ、プロジェクト立ち上げやタスク管理を半自動化する**ための仕組みを提供します。
新しい開発プロジェクトを開始する際に、

* ドキュメント作成
* マイルストーン設計
* イシュー登録
  などの初期作業を、AIのサポートを受けながら効率的に行えるように設計されています。

---

## 🏗️ 構成概要

### `.github/`

* **ISSUE_TEMPLATE/**
  一般的な開発プロジェクトで必要となるイシューテンプレートを収録。

  * `task.yml`：タスク登録用
  * `feature_request.yml`：機能提案用
  * `bug_report.yml`：バグ報告用
* **config.yml**
  テンプレート設定を統合管理。
* **COMMIT_TEMPLATE.txt**
  統一的なコミットメッセージフォーマットを定義。
* **GitHub Actions Workflow**
  feature_request イシューをトリガーに、AI連携プロンプト生成・自動イシュー作成を制御。

---

### `docs/`

システム開発プロジェクトにおける**各種ドキュメントひな型**を提供。

* `01_project_overview.md`：プロジェクト概要書
* `02_requirements_definition.md`：要件定義書
* `03_auth_design.md`：認証・認可設計書
* `04_system_architecture.md`：システム構成図
  これらはChatGPT等に読み込ませることで、**プロジェクト要件の自動深掘り・定義支援**を行える設計。

---

### `ai_helper/`（AI連携支援ブランチ）

AIとの連携をサポートするための補助フォルダ。
主に`ai_helper`ブランチで管理。

```
ai_helper/
 ├─ README.md                     # AI支援機能の概要説明
 ├─ templates/                    # プロンプトやYAMLテンプレート
 ├─ feature_requests_proposal/    # AI提案生成プロンプトの出力先
 ├─ auto_issue_builder/           # AI出力のyamlを格納し、自動イシュー生成に使用
 └─ scripts/                      # GitHub Actionsで利用されるPythonスクリプト
```

---

## ⚙️ GitHub Actionsによる連携フロー

1. **開発者が Feature Request Issue を登録**

   * GitHub Actions が起動し、
     現在のマイルストーン・Openタスク一覧・イシュー内容を取得。
2. **AI連携用プロンプトを自動生成**

   * テンプレートに流し込み、`ai_helper/feature_requests_proposal/`に出力・コミット。
3. **開発者がAIに依頼**

   * 生成されたプロンプトをChatGPTなどに渡し、AIから提案（YAML形式）を受け取る。
4. **提案内容を反映**

   * AI出力YAMLを`ai_helper/auto_issue_builder/`に保存・push。
5. **GitHub Actionsが実行**

   * YAML定義に基づき、マイルストーン・イシューを自動作成。
